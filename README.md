AI Landing Page Personalizer
Problem:
Users click ads with specific intent but landing pages often don’t match, reducing conversions.
Solution:
This tool analyzes ad creatives and aligns landing page messaging using AI to improve
conversions.
How It Works:
1. User inputs ad + URL
2. System analyzes ad intent
3. AI rewrites headline and CTA
4. Shows personalized output
Key Components:- Ad Analyzer- Page Parser- Personalization Engine
Tech Stack:- Python- Streamlit- Google Gemini API
